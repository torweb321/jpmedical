## 2. 产品矩阵：两条业务线

### 2.1 业务线全景

```
                 启航医疗
               /          \
         预防体检        第二诊疗意见+治疗
       （健康人群）       （已确诊患者）
       价格：1.1~5万RMB     价格：2.1~15万RMB
       交付：赴日1天         交付：远程/赴日
```

### 2.2 为什么二诊是核心入口？

第二诊疗意见虽然不能广撒网广告，但一旦成交：

| 阶段 | 产品 | 价格 | 利润 |
|------|------|------|------|
| 第一步 | 视频二诊（远程） | 45~66万日元 | ✅ 中 |
| 第二步（如需治疗） | 赴日治疗协调（基础） | 66万日元 | ✅ 高 |
| 第三步（长期治疗） | 治疗协调（标准/VIP） | 110~330万日元 | ✅ 最高 |

### 2.3 获客策略差异

| | 预防体检 | 第二诊疗意见 |
|--|---------|------------|
| **能不能广告推？** | ✅ 可以（健康管理类内容） | ❌ 不能（医疗广告限制） |
| **获客方式** | 小红书/知乎/朋友圈/SEO | 病友群/转介绍/体检客户转化/SEO |
| **决策周期** | 1~4周 | 1~3天（紧迫性强） |
| **客单价** | 1.1~5万RMB | 2.1万RMB起 |
| **转化率** | 中等 | 低（受众窄但意向强） |

### 2.4 最佳销售路径

```
体检客户 → 发现异常 → 推荐二诊 → 确诊 → 赴日治疗
```

体检客户是二诊的天然来源——拿到报告发现有结节、指标异常，最需要找日本专家确认。**每成交一个体检客户，都可能转化成一个二诊+治疗客户。**

## 5. 第二诊疗意见（视频二诊）深度解读

### 5.1 产品概述

第二诊疗意见（Second Opinion）是指患者听取主治医生意见后，再听取其他医院/专家的意见。

**核心价值：** 减少误诊、找到最适合的治疗方案、加深对治疗的理解。

**关键限制：** 不能广告推广（医疗广告法限制），只能通过精准渠道获客。

### 5.2 价格体系

| 服务模式 | 价格（日元） | 约合RMB | 交付物 |
|---------|------------|---------|--------|
| 远程视频咨询 | 450,000~660,000 | ≈21,000~31,000 | 视频面诊+口译，无书面报告 |
| 来院咨询 | 450,000~660,000 | ≈21,000~31,000 | 来日本面诊+口译 |
| 书面代理咨询 | 450,000~660,000 | ≈21,000~31,000 | EAJ整理问答书面报告 |

> ※ 以上费用为服务费，不含交通、住宿等费用。可根据客户需求定制方案。

**可选追加：**
- 病理诊断（国立癌症研究中心中央医院）：+165,000日元

### 5.3 目标客户

| 特征 | 说明 |
|------|------|
| **已确诊** | 癌症、心脑血管疾病等重大疾病 |
| **对治疗方案不确定** | 医生建议手术/化疗，想确认是否最优 |
| **担心误诊** | 检查发现异常，想找日本专家二次确认 |
| **想了解日本治疗** | 考虑赴日治疗，先通过二诊评估可行性 |

### 5.4 完整流程

```
01 初期咨询 → 02 提交资料 → 03 付费
→ 04 病历翻译+预约专家 → 05 视频咨询/书面代理
→ 06 报告翻译+反馈
```

- 服务申请表（由我们提供）

## 8. 客户常见问题 & 标准回答

### Q1: 和国内高端体检有什么区别？

> 国内高端体检通常套餐价3,000~10,000元，但关键区别不在价格，在于：
>
> 1. **早期发现能力**：日本精密体检追求发现2~5mm的早期癌，国内普通体检到1cm以上才能发现。早期胃癌治愈率超过95%，晚期不到20%。
>
> 2. **设备和医生**：日本医院由专业医生亲自操作和读片（比如胃镜是消化科医生做，不是技师），设备是PET-CT、3T-MRI等尖端机型。
>
> 3. **服务体验**：每天限制人数，一对一私密检查，医生详细解说结果，全程隐私保护。不是流水线。

### Q2: 需要多少天？怎么安排行程？

> - **体检本身：** 大部分套餐1天完成。肠镜套餐2天。
> - **建议行程：** 3天2晚。第1天飞到日本休息（医院规定体检前一天必须到达日本），第2天体检，第3天返程。
> - **请假时间：** 3~4天足够（含路途）。

### Q3: 价格包含什么？不包含什么？

> **包含：**
> - 体检预约调整费 + 体检套餐费
> - 全程中文医疗翻译陪同
> - 中文问诊表 + 体检报告中文翻译
> - 报告国际EMS邮寄
>
> **不包含：**
> - 国际机票 + 日本住宿 + <strong>餐饮费用</strong>（部分套餐含简餐除外）
> - 追加项目（如MRI、PET-CT、肠镜等）
> - 个人消费

### Q4: 不会日语怎么办？

> 完全不用担心。我们提供全程中文医疗翻译陪同：
> - 体检前：中文问诊表填写指导
> - 体检当天：翻译全程陪同，从报到到检查到医生解读
> - 体检后：日文原版报告 + 中文翻译版寄送到家

### Q5: 取消/改期政策？

> 各医院略有不同，参考标准：
> - 体检15~90天前取消：收10~30%
> - 体检8~14天前取消：收20~30%
> - 体检2~7天前取消：收30~50%
> - 体检1天前取消：收80%
> - 体检当天取消：收100%
> - 迟到1小时以上 = 当天取消
>
> **建议：** 确定行程再预约，不要轻易改期。

### Q6: 体检报告多久能拿到？

> - **当场出的结果：** 超声、CT、心电图等，医生当天解读
> - **血液等分析项目：** 约4~6周出完整报告
> - **报告形式：** 日文原版 + 中文翻译版，EMS寄送

### Q7: 体检发现异常怎么办？

> - 圣路加等医院有后续诊疗通道，需要治疗的可以转到本院治疗（以医院判断为准）

> - 但请注意：体检不是保险，不能保证100%发现所有疾病

### Q8: 我比较年轻（30多岁），需要去吗？

> 如果您有癌症家族史、经常熬夜/应酬/高压工作，或者就是想要一份"放心"，30多岁也有价值。日本体检的优势在于发现早期病变——年轻一样可能得病，早期发现治愈率完全不同。
